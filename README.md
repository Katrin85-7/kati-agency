# kati-agency

Landing for my agency — **Katie Agency**, AI Media Agency нового поколения.

## Сайт

Одностраничный лендинг для коучей, психологов, наставников и экспертов мягких ниш.

| Файл / папка | Назначение |
|---|---|
| `index.html` | Главная страница |
| `assets/` | Изображения бренда |
| `netlify.toml` | Конфиг деплоя на Netlify |
| `404.html` | Редирект на главную при 404 |
| `docs/netlify-deploy.md` | Пошаговая инструкция Netlify |
| `push-to-github.ps1` | Пуш в GitHub |

## Онлайн

**GitHub Pages:** https://katrin85-7.github.io/kati-agency/

**Netlify:** см. [docs/netlify-deploy.md](docs/netlify-deploy.md) — подключи репозиторий, publish = `.`, сборка не нужна.

## Локально

Открой `index.html` в браузере или разверни на статическом хостинге (GitHub Pages, Vercel, Netlify).
